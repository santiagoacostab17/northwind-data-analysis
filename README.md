# Northwind Sales Analysis Project

## Overview
This project analyzes sales data from the Northwind database using SQL and Power BI to identify business trends and key performance indicators.

## Tools
- MySQL
- Power BI

## Key KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Order Value (AOV)

## Key Insights
- Beverages and Seafood are the top revenue categories.
- Sales do not follow a clear pattern
- The sales capacity of employees appears to vary linearly.
